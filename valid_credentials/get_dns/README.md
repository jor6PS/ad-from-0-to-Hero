# Obtener DNS ip

```Bash
# Obtencion del DNS con la herramienta adidnsdump sacada de Github: https://github.com/dirkjanm/adidnsdump
python adidnsdump/dnsdump.py -u '<nombre completo dominio>\<user> -p '<pass>' <nombre dns del DC>

```
